54 /* <Description> */
55 /* Computes the exact bounding box of an outline. This is slower */
56 /* than computing the control box. However, it uses an advanced */
57 /* algorithm which returns _very_ quickly when the two boxes */
58 /* coincide. Otherwise, the outline Bezier arcs are walked over to */
59 /* extract their extrema. */
60 /* */
61 /* <Input> */
62 /* outline :: A pointer to the source outline. */
63 /* */
64 /* <Output> */
65 /* abbox :: The outline's exact bounding box. */
66 /* */
67 /* <Return> */
68 /* FreeType error code. 0 means success. */
69 /* */
