Description détaillée

SOURCE PROCESSOR CLASS.

The SourceProcessor is in charge or reading a C source file and decomposing it into a series of different "SourceBlock" objects.

each one of these blocks can be made of the following data:

  • A documentation comment block that starts with "/**" and whose exact format will be discussed later
  • normal sources lines, include comments
